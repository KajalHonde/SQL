SQL Practice Repository

This repository contains a collection of SQL practice scripts designed to build and strengthen core SQL concepts using MySQL.

Contents

Basic SQL commands and syntax

Comparison, logical, and arithmetic operators

Data Manipulation Language (INSERT, UPDATE, DELETE)

Sorting and filtering data

ENUM and SET data types

Practice queries using sample datasets

Tools

MySQL

MySQL Workbench

Purpose

This project is intended for learning, practice, and revision of SQL fundamentals for students, beginners, and interview preparation.
